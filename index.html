Tecnologías y frameworks

  • VS Code

    Múltiples herramientas y extensiones disponibles hacen que mi experiencia desarrollando sea cómoda y eficaz.

  • Node.js

    Un potente entorno de desarrollo que funciona mediante el lenguaje JavaScript, comúnmente empleado para backend.

  • HTML

    El lenguaje de marcado para la estructuración de cualquier sitio web.

  • CSS

    El lenguaje para definir la presentación y diseño visual de los sitios web.

  • Bootstrap

    Una framework de diseño basado en CSS que facilita la creación de sitios web responsivos de manera rapida.

  • Laravel

    Framework de desarrollo que funciona con el lenguaje PHP y ofrece múltiples ventajas sobre el desarrollo con PHP puro.

  • Angular

    Framework de desarrollo para construir aplicaciones web altamente escalables. Organiza la aplicación en componentes reutilizables y emplea principalmente TypeScript, HTML y CSS

  • MongoDB

    Un gestor de bases de datos no relacional (NoSQL) que ofrece funciones potentes para la gestión y manipulación de documentos.

  • MySQL

    Gestor de bases de datos relacional (SQL) que emplea el lenguaje de consulta estructurado. Ofrece alta eficiencia y estabilidad.

  • Git

    Una herramienta de control de versiones muy útil, empleada para gestionar el historial y las diferentes versiones de un proyecto durante su desarrollo.
